---
title: "Async concurrency"
description: "Concurrency and coordination helpers from bytekit/async."
---

## Exports

- `parallel`
- `sequential`
- `race`
- `allSettled`

## Parallel example

```ts
import { parallel } from "bytekit/async";

const results = await parallel(
  jobs.map((job) => () => runJob(job)),
  { concurrency: 3 }
);
```

## Race example

```ts
import { race, sleep } from "bytekit/async";

const result = await race([
  fetchPrimary(),
  fetchSecondary(),
  sleep(5000).then(() => {
    throw new Error("Timeout");
  }),
]);
```
