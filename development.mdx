## Prerequisites

- Node.js 18 or newer
- A global or local `mint` CLI installation

## Run the docs locally

```bash
npm i -g mint
mint dev
```

Mintlify serves the site locally, usually at `http://localhost:3000`.

## Check for broken links

```bash
mint broken-links
```

Run this before you open a pull request or publish a larger docs change.

## Update site configuration

Most project-level configuration lives in `docs.json`.

- Update the site name, colors, navbar links, and footer links there.
- Keep page slugs aligned with the `navigation` entries.
- Prefer moving or renaming pages together with their `href` references.

## Writing guidance for this repo

- Use active voice and second person.
- Keep sentences short.
- Use bold text for UI labels.
- Use code formatting for commands, paths, import paths, and identifiers.
